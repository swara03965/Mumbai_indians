# Chennai Super Kings Fan Hub 🦁

A comprehensive website dedicated to Chennai Super Kings (CSK), one of the most successful IPL teams. This fan hub serves as a central platform for fans to access team-related information, match schedules, player profiles, statistics, news updates, and engage with fellow fans.

## Features 🚀

### 🏠 Home Section
- **Hero Banner**: Eye-catching introduction with team statistics
- **Team Overview**: Quick facts about CSK's achievements
- **Call-to-Action**: Direct navigation to key sections

### 📅 Match Schedule
- **Upcoming Matches**: View fixtures with dates, times, and venues
- **Completed Matches**: Check recent results and match summaries
- **Tab Navigation**: Easy switching between upcoming and completed matches
- **Interactive Cards**: Hover effects and detailed match information

### 👥 Player Profiles
- **Star Players**: Profiles of key CSK players
- **Player Statistics**: Runs, wickets, averages, and career highlights
- **Professional Images**: High-quality player photographs
- **Responsive Grid**: Optimized layout for all devices

### 📊 Team Statistics
- **IPL Titles**: Display of championship wins
- **Win Percentage**: Team performance metrics
- **Records**: Highest scores and notable achievements
- **Fan Following**: Social media and engagement statistics

### 📰 News & Updates
- **Latest News**: Recent articles and match reports
- **Categorized Content**: Match reports, team news, interviews
- **Share Functionality**: Easy sharing on social media
- **Responsive Cards**: Mobile-friendly news display

### 🎉 Fan Zone
- **Interactive Polls**: Vote for your favorite players
- **Social Media Integration**: Links to official CSK social accounts
- **Fan Engagement**: Features to connect with the community
- **Real-time Results**: Live poll results with percentage display

## Technology Stack 💻

- **HTML5**: Semantic markup for accessibility
- **CSS3**: Modern styling with animations and transitions
- **JavaScript ES6+**: Interactive features and dynamic content
- **Responsive Design**: Mobile-first approach with media queries
- **Local Storage**: Persistent poll data and user preferences

## Getting Started 🚀

1. **Clone or Download**: Get the project files
2. **Open in Browser**: Launch `index.html` in your preferred browser
3. **Or Use Local Server**: Run `python3 -m http.server 8000` for better experience
